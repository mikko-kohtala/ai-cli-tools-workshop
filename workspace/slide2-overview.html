8-Week Program Structure

Weeks 1-2: Shared Foundation

LLM fundamentals, agentic patterns, tool setup, project configuration

Week 3: Test Automation Fundamentals

Test generation patterns, AI-assisted test writing, coverage analysis

Week 4: Core QA Workflows

Regression testing, API testing, performance analysis, bug documentation

Week 5: Advanced Testing Techniques

E2E orchestration, visual regression, test maintenance, flaky tests

Week 6: QA Integration

Developer collaboration, CI/CD integration, quality metrics, evangelism

Weeks 7-8: Shared Capstone

Building agents with Claude Agent SDK, capstone project, deployment

Outcome: Ship higher quality software with AI-assisted testing
