# QA Learning Path

**AI-Assisted Test Automation & Quality Assurance**

8-Week Comprehensive Curriculum

**Tools:** Playwright, Cypress, Claude Code, Codex CLI

**Outcome:** Ship higher quality software with AI-assisted testing

---

## Program Structure

| Weeks | Focus | Type |
|-------|-------|------|
| 1-2 | Foundations & AI Basics | Shared |
| 3-6 | QA-Specific Skills | Divergent |
| 7-8 | Building Agents & Capstone | Shared |

---

## Week 1: Foundations & Mindset
*Understanding AI, agents, and the right mental model*

### Objectives
- Understand what LLMs are and are not good at
- Learn what makes AI "agentic" vs simple chat
- Install and configure Claude Code
- Set up project configuration (CLAUDE.md)
- Understand safety: approval modes and sandboxing

### Daily Topics

| Day | Topic | Description |
|-----|-------|-------------|
| 1 | What Are LLMs? | Understanding language models, tokens, context windows, and capabilities |
| 2 | Agentic vs Chat | The difference between chat interfaces and agentic tool-using systems |
| 3 | Tool Setup | Installing and configuring Claude Code for your development workflow |
| 4 | Project Configuration | Setting up CLAUDE.md and project-specific AI configuration |
| 5 | Safety & Boundaries | Approval modes, sandboxing, and establishing safe AI workflows |

**Tools:** Claude Code

**Milestone:** Development environment configured with initial project setup

---

## Week 2: Planning & Agentic Patterns
*Workflows, planning, and when to use agents*

### Objectives
- Understand workflows vs autonomous agents
- Learn the 5 core agentic workflow patterns
- Master Claude Code's planning mode
- Understand ReAct: reasoning before acting
- Start core project: scaffold and architecture

### Daily Topics

| Day | Topic | Description |
|-----|-------|-------------|
| 1 | Workflows vs Agents | When to use deterministic workflows versus autonomous agent behavior |
| 2 | Agentic Patterns | The 5 core patterns: prompt chaining, routing, parallelization, orchestration, evaluation |
| 3 | Planning Mode | Using Claude Code's planning mode (Shift+Tab) for complex tasks |
| 4 | ReAct Pattern | Reasoning and acting: how agents think before they act |
| 5 | Project Kickoff | Starting your core project with AI-assisted scaffolding and architecture |

**Tools:** Claude Code

**Milestone:** Core project scaffolded with architecture decisions documented

## Week 7: Building Agents
*Creating production-ready agents with Claude Agent SDK*

### Objectives
- Set up and understand the Claude Agent SDK
- Build custom tools for your agents
- Implement agent loops with context management
- Create multi-agent systems with orchestration
- Add safety guardrails and approval workflows

### Daily Topics

| Day | Topic | Description | Tasks |
|-----|-------|-------------|-------|
| 1 | Agent SDK Setup | Installing and understanding the Claude Agent SDK architecture | Install Agent SDK, Understand core concepts |
| 2 | Custom Tools | Building tools that agents can use: file operations, APIs, databases | Create custom tools, Define tool schemas |
| 3 | Agent Loops | Implementing robust agent loops with automatic context management | Build agent loop, Handle context overflow |
| 4 | Multi-Agent Systems | Orchestrating multiple agents for complex workflows | Design multi-agent architecture, Implement agent handoffs |
| 5 | Safety & Guardrails | Adding approval workflows, rate limits, and safety boundaries | Implement guardrails, Add human-in-the-loop approval |

**Tools:** Claude Agent SDK, Claude Code
